Proiect IntelliJ: L06_Pachete_Exceptii_Asertiuni

Cum rulezi:
1. Deschide IntelliJ IDEA.
2. File -> Open.
3. Alege folderul: L06_Pachete_Exceptii_Asertiuni
4. In Project Structure seteaza SDK Java daca cere.
5. Deschide fisierul .java din src si ruleaza clasa care are main().

Din terminal:
cd src
javac *.java
java Test / java NumeleClasei

Pentru Exercitiul03 exista pachetul myPackage.
Rulare din terminal:
cd src
javac myPackage/*.java *.java
java Exercitiul01
java Exercitiul02
java Exercitiul03
etc.
