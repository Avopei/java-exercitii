Rezolvari L08 - Java I/O

Codul este scris simplu, in stil de laborator:
- BufferedReader / InputStreamReader
- FileReader / FileWriter
- FileInputStream / FileOutputStream
- StreamTokenizer
- SequenceInputStream
- ObjectOutputStream / ObjectInputStream
- RandomAccessFile nu este necesar aici, dar stilul este asemanator exemplelor.

Pentru rulare:
1. Copiaza continutul unui fisier TXT intr-un fisier .java cu acelasi nume al clasei.
2. Compileaza:
   javac NumeFisier.java
3. Ruleaza:
   java NumeFisier

Unele exercitii au nevoie de fisiere de intrare, mentionate in comentarii.
