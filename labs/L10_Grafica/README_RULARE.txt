Rezolvari L10_SE - Java grafica AWT

Fiecare fisier .txt contine cod Java. Pentru rulare:
1. Copiaza continutul unui fisier .txt intr-un fisier .java cu acelasi nume ca public class.
   Exemplu: Ex01_NumeGrupa.txt -> Ex01_NumeGrupa.java
2. Compileaza:
   javac Ex01_NumeGrupa.java
3. Ruleaza:
   java Ex01_NumeGrupa

Pentru exercitiul 3, fisierul image_01.jpg trebuie sa fie in acelasi folder cu Ex03_ImagineFisier.java.
La exercitiul 1 schimba valorile:
String nume = "Numele tau";
String grupa = "Grupa ta";
cu numele si grupa ta.
