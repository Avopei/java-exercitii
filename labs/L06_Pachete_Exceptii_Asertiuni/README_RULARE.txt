Rezolvari L06 - Packages, Exceptions handling, Assertions

Fiecare exercitiu este intr-un folder separat.
Codul este scris simplu, in stilul exemplelor din laborator: Scanner, arrays, try/catch/finally, throw/throws, clase simple si pachete.

Comenzi de rulare:

cd ex1
javac Exercitiul01.java
java Exercitiul01
